📊 Metodologia e Dados

🔍 Fonte dos Dados
Dados extraídos dos registros oficiais de votação nominal da Câmara dos Deputados para os dois turnos da PEC da Bandidagem.
⚡ Processamento
Análise computacional usando Python e D3.js. Dados validados por métodos independentes para garantir precisão de 95%+ em todas as estatísticas.
📈 Métricas
Índice Rice de coesão partidária, correlação de Pearson, análise de fluxo e estatísticas descritivas para padrões regionais.
✅ Validação
Verificação cruzada independente de todos os dados. Relatório completo de validação disponível demonstrando acurácia para publicação.
